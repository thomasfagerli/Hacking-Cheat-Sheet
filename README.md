## Website enumeration

### FFUF

Ffuff can be used to scan for directories, files, extensions, vhosts, parameter values. 
Assign keyword to wordlist:
'-w /usr/share/SecLists/Discovery/Web-Content/directory-list-2.3-small.txt:FUZZ'
